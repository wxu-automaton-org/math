抽象

抽象是直接抽取自然界事物在某个条件下的本质特征，为了解决自然界的问题，如：一个苹果、两个苹果、三个苹果 可以抽象为 1, 2, 3，我们数苹果的时候就不必大费口舌说 x个苹果
=> 正整数诞生
又如：一个苹果和另一个苹果混在一起 可以抽象为 1+1（或者说这是一种约定）
=> 正整数的加法诞生
又如：我原本有三个苹果，但失去了四个苹果 可以抽象为 3-4
=> 正整数减法诞生
又如：一个人欠了我五个苹果 可以抽象为 -5
=> 负整数诞生
==> 整数体系，整数上的运算逐步完善

整数及其运算诞生 => 其他数系诞生
